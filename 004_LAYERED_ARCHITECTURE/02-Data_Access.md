# 내용

## 학습 키워드

- DAO(Data Access Object)
  - Layer Architecture에서 언급한 3티어 방식에서 data layer에서 사용
  - 데이터 접근을 위해 사용하는 개체이다.
- List
  - 선형 데이터를 저장하는 인터페이스
- Map
  - key,value 데이터를 저장하는 인터페이스.
- DAO라는 인터페이스를 만들고, List를 저장하고 받는 DTO나 Map으로 데이터를 관리하는 DAO오 같은 구현체로 구성할 수 도 있다.
