# 키워드

## 학습 키워드

- URI & URL & URN
- MIME type

# 내용

- URI : Unifom Resource Identifier
  - 식별자. 웹의 리소스를 식별한다.
  - URL이랑 거의 동의어. URN 잘 안쓰는 표현.
- MIME type: 요청이나 응답의 포맷을 표현함.
  - text/plain : 이메일에서 사용
  - text/html : 웹문서
  - application/xml : 범용이다.
  - etc..

# 필기

- 웹의 리소스를 식별하는 방법, URI
- MIME : 요청과 응답의 포맷관련된 용어.
- 다음 시간엔 논문엔 없는 콜렉션 패턴까지 할 것이다.
